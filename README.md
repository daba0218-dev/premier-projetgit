# 🎧ISI-Stream
# Avec l'equipe Code-queens👑

##  Description
ISI-Stream est un projet de lecteur de musique réalisé en HTML et CSS.

Le projet contient :
- Une page d'accueil avec une liste d'artistes (sous forme de cartes avec images)
- Une page de lecture avec l'image de l'artiste et un lecteur audio

##  Objectifs
- Créer une interface moderne inspirée de Spotify / Deezer
- Utiliser HTML et CSS pour le design
- Simuler un lecteur audio

## Fonctionnalités
- Liste de 10 à 15 artistes
- Un artiste cliquable
- Navigation vers une page de lecture
- Affichage de l’image de l’artiste
- Lecture d’un audio
- Boutons (play, pause, suivant, précédent)

##  Technologies utilisées
- HTML
- CSS

## Structure du projet
- index.html → liste des artistes
- artiste.html →  page lecteur
- style.css → design
- images/ → contient les images
- audio/ → contient les sons
